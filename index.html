História do HTML

O HTML (HyperText Markup Language) é a linguagem fundamental utilizada para criar páginas da web. Sua história remonta ao início da World Wide Web nos anos 1990.

Anos 1980: Tim Berners-Lee, um cientista da computação britânico, desenvolveu a ideia de um sistema de informações conectado globalmente enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Ele propôs o conceito de hipertexto, que permitia aos usuários navegar por informações interconectadas.

Anos 1990: Berners-Lee cria o primeiro navegador, o WorldWideWeb (mais tarde renomeado Nexus), e o primeiro servidor web. Ele também desenvolveu o HTML como uma linguagem de marcação simples para criar e vincular documentos hipertexto.

Anos 1993: O HTML evoluiu para a versão 2.0, que introduziu elementos como tabelas e formulários. No entanto, o desenvolvimento estava fragmentado, com diferentes navegadores interpretando o HTML de maneira diferente.

Anos 1995: O HTML 3.0 foi lançado com mais recursos de formatação e layout, mas a inconsistência na interpretação pelos navegadores continuou a ser um problema.

Anos 1997: O HTML 4.0 foi lançado com recursos avançados de layout e formatação, introduzindo a noção de folhas de estilo em cascata (CSS) para controlar a aparência das páginas. Isso separou a estrutura (HTML) da apresentação (CSS).

Anos 2000: A web continuou a crescer exponencialmente, mas a falta de padrões consistentes entre os navegadores levou a problemas de compatibilidade. Isso resultou em práticas de desenvolvimento complicadas e incompatibilidades entre sites.

Anos 2000 a 2010: Organizações como o World Wide Web Consortium (W3C) trabalharam para desenvolver padrões mais robustos, incluindo o HTML5, que foi projetado para ser mais flexível, suportar multimídia (como áudio e vídeo) nativamente e melhorar a semântica dos documentos web.

Anos 2014: O W3C oficialmente recomendou o HTML5 como um padrão web, e ele se tornou amplamente adotado por desenvolvedores e navegadores. O HTML5 trouxe tags semânticas (como header, nav, section, etc.) que melhoraram a acessibilidade e a estruturação das páginas.

Presente: O HTML5 continua sendo o padrão dominante para a criação de páginas da web. O W3C e a comunidade de desenvolvedores continuam a trabalhar em melhorias e extensões para a linguagem, visando melhorar a experiência do usuário e a acessibilidade.

Curiosidade

Topo

A Web como o mundo conhece hoje não era a ideia inicial de quem participou da sua criação, tampouco de Tim Berners-Lee, conhecido como o Pai da World Wide Web (o tal do WWW). O primeiro site da grande rede é um belo exemplo dessa distinção de finalidade — e, hoje (6), este marco histórico completa 30 anos desde a sua estreia.

The Project, como é conhecida o primeiro site da história, foi criado pelo próprio Berners-Lee. Sua estreia aconteceu no dia 6 de agosto de 1991 e, nele, seu criador descreve brevemente detalhes da World Wide Web.

O objetivo deste espaço principal era permitir que os profissionais do Centro de Europeu de Pesquisa Nuclear (CERN) — no qual Berners-Lee atuava como físico — tivessem acesso rápido aos códigos e procedimentos para criar sites semelhantes.

Basicamente, The Project era constituído por um bloco de HTML simples, sem recursos de estilização, imagens ou qualquer recurso dinâmico. A página era recheada de links que encaminham visitantes para respostas rápidas a perguntas como “o que existe por aí?”, e “como posso ajudar?”.

O domínio do site ainda é o mesmo e funciona perfeitamente — e você pode acessar aqui o primeiro site da Web. Hoje, qualquer pessoa com o mínimo de conhecimento em linguagem de marcação de hipertexto é capaz de fazer um projeto semelhante usando o Bloco de Notas do Windows, por exemplo. A título de curiosidade, você pode experimentar o “Inspecionar elemento” ou apertar a tecla F12 para abrir os detalhes da formatação.

Tags do HTML

Topo

No HTML, as "tags" são elementos fundamentais que definem a estrutura e o conteúdo de uma página da web. Elas são usadas para marcar partes específicas do texto e fornecer instruções ao navegador sobre como renderizar e exibir o conteúdo. As tags são escritas entre colchetes angulares ("<" e ">") e geralmente vêm aos pares, consistindo em uma tag de abertura e uma tag de fechamento correspondente.

Tags de Abertura e Fechamento

As tags geralmente vêm em pares, onde a primeira tag é a de abertura e a segunda é a de fechamento. A tag de fechamento tem uma barra ("/") antes do nome da tag. Por exemplo, <p> é uma tag de abertura para um parágrafo, e </p> é a tag de fechamento correspondente.

Elementos e Conteúdo

As tags definem elementos em uma página, como parágrafos, títulos, imagens, links, listas, etc. O conteúdo que está entre as tags é o que será exibido na página.

Atributos

As tags podem conter atributos, que são informações adicionais sobre o elemento. Os atributos fornecem informações como o endereço de um link, o tamanho de uma imagem, a cor de um elemento, etc. Os atributos são inseridos dentro da tag de abertura e são compostos de um nome e um valor, separados por um sinal de igual ("=").

Tags de Auto-Fechamento

Algumas tags não têm uma tag de fechamento correspondente porque são auto-fechadas. Isso significa que elas não têm conteúdo interno. Exemplos são as tags <img>, <br>, e <input>.

Aninhamentos de Tags

As tags podem ser aninhadas umas dentro das outras para criar uma hierarquia de elementos. No entanto, é importante que elas sejam abertas e fechadas na ordem correta para evitar erros na renderização.

Semântica

O HTML5 introduziu muitas tags semânticas que dão significado ao conteúdo, facilitando para os motores de busca e leitores de tela entenderem a estrutura do documento. Exemplos incluem <header>, <nav>, <main>, <article>, <footer>, entre outros.

Exemplos de Algumas Tags e seus Significados

Topo

<h1>: É a tag de cabeçalho de nível mais alto e geralmente é usada para o título principal da página. Ela é a mais visível e tem o maior tamanho em comparação com as outras tags de cabeçalho.

<h2>: É uma tag de cabeçalho de nível um pouco menor que o h1. Geralmente é usado para títulos de seções importantes após o título principal.

<h3> até h6>: São tags de cabeçalho de níveis subsequentes, cada uma com um tamanho menor e indicando níveis de importância decrescentes. Elas são usadas para estruturar o conteúdo em subseções e detalhes dentro das seções principais.

<p>: Ela é usada para definir um parágrafo de texto. Tem <p> (abertura) e </p> (fechamento)

<mark>: é usada para indicar que um trecho de texto dentro de um parágrafo ou outro elemento deve ser realçado de alguma forma, normalmente com uma cor de fundo. Tem <mark> (abertura) e </mark> (fechamento)

<small>: é uma tag do HTML que é usada para indicar que o texto dentro dela deve ser renderizado em um tamanho menor do que o texto circundante. Tem <small> (abertura) e </small> (fechamento)

<i>: é usada para indicar texto em itálico. Tem <i> (abertura) e </i> (fechamento)

<u>: é usada para sublinhar o texto, indicando que ele deveria ser exibido com uma linha horizontal sob eleTem <u> (abertura) e </u> (fechamento)

<strong>: Essa tag é usada para enfatizar o texto, geralmente exibindo-o em negrito. Ela indica que o texto é importante ou deve ser destacado. Tem <strong> (abertura) e </strong> (fechamento)

<ol>: Essa tag é usada para criar uma lista ordenada, que exibe itens numerados sequencialmente. Tem <ol> (abertura) e </ol> (fechamento)

<ul>: Essa tag é usada para criar uma lista não ordenada, que exibe itens com marcadores. Tem <ul> (abertura) e </ul> (fechamento)

<li>: Essa tag é usada dentro de <ol> e <ul> para definir cada item da lista, seja ordenada ou não ordenada.Tem <li> (abertura) e </li> (fechamento)

<a>: Essa tag é usada para criar links. Você define o atributo href para especificar o URL de destino do link. Tem <a> (abertura) e </a> (fechamento)

<hr>: Essa tag cria uma linha horizontal para separar o conteúdo. Geralmente é usada para dividir seções distintas. Não tem tag de fechamento.

<sub>: Essa tag é usada para exibir texto como subscrito, ou seja, ligeiramente abaixo da linha base. É frequentemente usado para fórmulas químicas ou notações matemáticas. Tem <sub> (abertura) e </sub> (fechamento)

<sup>: Essa tag é usada para exibir texto como sobrescrito, ou seja, ligeiramente acima da linha base. Também é usado para fórmulas e notações. Tem <sup> (abertura) e </sup> (fechamento)

<blockquote>: Essa tag é usada para destacar um bloco de texto que é uma citação de outra fonte. O texto é frequentemente recuado e pode ser usado para melhorar a formatação de citações. Tem <blockquote> (abertura) e </blockquote> (fechamento)

<font>: Embora tenha sido amplamente usada para definir o tamanho, a cor e o tipo de fonte, ela é considerada obsoleta. É recomendável usar CSS para estilos de fonte. Tem <font> (abertura) e </font> (fechamento)

<del>: Essa tag é usada para indicar texto que foi removido ou riscado. É frequentemente usado em edições de documentos. Tem <del> (abertura) e </del> (fechamento)

<abbr>: Essa tag é usada para definir uma abreviação ou acrônimo no texto. O atributo title é usado para fornecer a expansão completa quando o usuário paira sobre o texto. Tem <abbr> (abertura) e </abbr> (fechamento)
